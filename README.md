# DHNLNtupleAnalysis

`DHNLNtupleAnalysis` is a framework for reading Ntuples generated by [DHNLAlgorithm](https://gitlab.cern.ch/atlas-phys/exot/ueh/EXOT-2017-19/DHNLAlgorithm), developed for the displaced heavy neutral lepton (DHNL) analysis. For installation instructions, see the "Getting Started" section below.

## Getting Started

To clone the project: 

```
setupATLAS
lsetup git
git clone ssh://git@gitlab.cern.ch:7999/atlas-phys/exot/ueh/EXOT-2017-19/DHNLNtupleAnalysis.git
```

For a new commit: 

```
git add . 
git commit -m "description of my change"
git push -u origin master
```

To run the code: 

```
cd python 
python run.py 
```
